## Install

```
pip install -r requirements.txt
```

## Start

### Windows: 

**CLI**:
```
py ./mcbe_pack_decrypter.py
```

**UI**:
```
py ./with_ui/app.py
```

### MacOS/Linux: 

**CLI**:
```
python3 ./mcbe_pack_decrypter.py
```

**UI**:
```
python3 ./with_ui/app.py
```
